# Fashion Aggregation Platform

AI-powered fashion aggregation platform that scrapes and structures product data from multiple retailers.

## Current Features
- Zara scraper
- H&M scraper scaffold
- Structured product model
- Seed product pipeline

## Tech Stack
- TypeScript
- Node.js
- Web scraping utilities

## Vision
Building toward:
- Visual fashion search
- Cross-brand aggregation
- Recommendation systems
- AI outfit matching
